# Kuai

A multiplayer Chinese character typing game where players race to transcribe passages using Pinyin.

## What it is

Kuai presents Chinese text one character at a time. You type the Pinyin romanization and tone number (e.g. `hao3` for 好), press Enter to confirm, and Space to skip. Lessons are generated by AI and scale across difficulty levels. A 1v1 battle mode lets you challenge a friend to race through the same passage in real time.

## Tech stack

**Frontend** — React, TypeScript, Vite, Tailwind CSS, deployed on Vercel

**Backend** — Bun, Express, WebSockets (`ws`), deployed on Railway

**AI** — Google Gemini for lesson and passage generation

## Running locally

**Frontend**

```bash
cd kuai-frontend
npm install
npm run dev
```

**Backend**

```bash
cd kuai-server
bun install
bun run dev
```

The frontend runs on `http://localhost:5173` and the server on `http://localhost:3001`.

## Environment variables

**kuai-frontend** (`.env.local`)

```
VITE_GEMINI_API_KEY=
VITE_BATTLE_API_URL=http://localhost:3001
VITE_BATTLE_WS_URL=ws://localhost:3001/ws
```

## How battle mode works

When a host creates a room, the server generates a short room code and opens a WebSocket connection. The guest joins via a shared link. Once both players are connected, the server broadcasts a `GAME_START` message with an absolute `startsAt` timestamp. Each client schedules its countdown ticks against that timestamp independently, so both players enter the game at the same wall-clock moment regardless of network latency.

Progress updates (character index, WPM, score) are sent every 500ms and relayed to the opponent in real time. All room state is held in memory on the server and expires after 10 minutes.

## kuai-agents

A companion tool for generating Kuai lesson content. It runs a multi-agent pipeline powered by Gemini to process raw textbook vocabulary and produce stories at three difficulty levels.

**Running locally**

```bash
cd kuai-agents
npm install
npm run dev
```

Open `http://localhost:5173`, enter your Gemini API key, paste vocabulary text, and click "Run Agent Pipeline".

All story agents apply correct tone sandhi rules for Pinyin. The output is formatted code ready to paste directly into Kuai's lesson data.
